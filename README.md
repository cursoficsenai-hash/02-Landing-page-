# FonoCare - Landing Page Premium 2026

Uma landing page moderna, responsiva e de alta performance para uma clínica de fonoaudiologia, desenvolvida com tecnologias de ponta para 2026.

## 🚀 Tecnologias Utilizadas

- **HTML5** (Semântico)
- **Tailwind CSS** (Estilização Moderna e Premium)
- **Google Fonts** (Outfit & Plus Jakarta Sans)
- **FontAwesome** (Ícones)
- **Imagens Geradas por IA** (Contextualizadas para o negócio)

## ✨ Funcionalidades

- **Design Premium 2026:** Estilo Bento Grid, Glassmorphism e Mesh Gradients.
- **Interatividade:** Micro-animações suaves e transições elegantes.
- **Seções Estratégicas:**
  - Hero com CTA claro para agendamentos.
  - Grade de serviços detalhada.
  - Perfil profissional humanizado.
  - Prova social com depoimentos de pacientes.
  - Rodapé completo com integração de contatos.
- **Totalmente Responsivo:** Otimizado para todos os tamanhos de tela.

## 📁 Estrutura do Projeto

```text
├── assets/             # Imagens e recursos visuais
│   ├── hero.png        # Imagem principal da seção Hero
│   └── perfil.png      # Foto oficial da profissional
├── index.html          # Estrutura principal e estilos
└── README.md           # Documentação do projeto
```

## 🛠️ Como Visualizar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Se estiver usando o VS Code, recomendo a extensão **Live Server** para uma melhor experiência de desenvolvimento.
