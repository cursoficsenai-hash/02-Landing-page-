# Doutora Maria Eduarda | Fonoaudiologia e Cuidado

Uma Landing Page premium, moderna e totalmente responsiva desenvolvida para uma clínica de fonoaudiologia. Este projeto foi construído seguindo as melhores práticas de web design de 2026, focando em elegância, acolhimento e acessibilidade.

## ✨ Destaques do Projeto

- **Design Premium:** Uso de paleta de cores harmoniosa, tipografia moderna (Outfit e Inter) e efeitos visuais suaves.
- **Acessibilidade:** Inclui um recurso de leitura de conteúdo por voz (Narrador) integrado para melhorar a experiência do usuário.
- **Responsividade Total:** Adaptado para smartphones, tablets e desktops.
- **Interatividade:** Micro-animações, efeitos de hover e um botão flutuante para contato direto via WhatsApp.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e SEO otimizado.
- **CSS3 Puro:** Estilização avançada com variáveis CSS, Flexbox e Grid (sem frameworks externos como Tailwind, conforme solicitado).
- **JavaScript:** Lógica para o narrador de voz e interações de interface.
- **Phosphor Icons:** Biblioteca de ícones moderna e leve.

## 📁 Estrutura de Arquivos

```text
├── assets/images/      # Imagens do projeto (Hero, Perfil, etc.)
├── index.html          # Estrutura principal da página
├── style.css           # Estilização completa e design system
├── script.js           # Lógica de acessibilidade e interações
└── README.md           # Documentação do projeto
```

1. Clone o repositório:
